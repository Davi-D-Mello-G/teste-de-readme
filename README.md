# 1.  Descrição

O nosso projeto é um sistema baseado em plataformas utilizadas por instituições de ensino técnico, enfrentamos problemas em nossa instituição de ensino devido a má otimização da plataforma, a falta de funcionalidades e a difícil comunicação entre alunos e docentes - sendo necessário a utilização de plataformas externas para melhor interação online. A nossa proposta é a criação de um sistema base para todas as instituições de ensino técnico do país, que permita uma melhora significativa na comunicação entre docentes e alunos, possua interface intuitiva, acessibilidade para todos, calendário de eventos que pode ser atualizado pelos docentes e pela diretoria, e que seja acessada de forma prática e rápida.


---

# 2.  Objetivos

### Objetivo Geral
Desenvolver uma plataforma digital integrada que facilite a comunicação e o acesso às informações acadêmicas dentro das instituições de ensino técnico.

### Objetivos Específicos

- Desenvolver um sistema de gestão de usuários integrado
- Aplicar conceitos de desenvolvimento web e banco de dados
- Melhorar o processo de comunicação entre alunos, docentes e administração
- Criar uma interface simples, intuitiva e responsiva
- Centralizar informações acadêmicas em um único ambiente digital

---

# 3.  Público-Alvo

A plataforma é direcionada principalmente para:

- Administradores de TI
- Instituições de ensino técnico
- Gestores de tecnologia educacional
- Coordenadores administrativos
- Equipes responsáveis pela gestão de contas institucionais
- Docentes e alunos

---

# 4.  Funcionalidades

Entre as funcionalidades previstas no sistema estão:

- Cadastro de usuários
- Login e autenticação
- Registro de dados dos alunos (como notas, frequência e atividades)
- Painel de informações acadêmicas
- Calendário institucional de eventos
- Comunicação entre docentes e alunos
- Interface responsiva para diferentes dispositivos
